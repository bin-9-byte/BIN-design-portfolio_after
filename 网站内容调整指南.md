## 1. 字体调整

### 1.1 现有字体系统

项目使用了两种主要字体：
- **Cormorant Garamond**：用于衬线字体（font-serif），主要用于标题
- **Montserrat**：用于无衬线字体（font-sans），主要用于正文和标签

中文字体：
- **Source Han Sans SC**：思源黑体，用于中文无衬线字体
- **YuanLiu MingTi**：源流明体，用于中文衬线字体

### 1.2 修改字体

#### 修改英文字体

1. 在 `public/fonts/fonts.css` 文件中添加新字体的 @font-face 定义：

```css
/* 新字体 */
@font-face {
  font-family: 'Your New Font';
  font-style: normal;
  font-weight: 400;
  src: url('/fonts/static/YourNewFont-Regular.ttf') format('truetype');
}
```

2. 在组件中修改字体类名：
   - 将 `font-serif` 替换为新的衬线字体类名
   - 将 `font-sans` 替换为新的无衬线字体类名

#### 修改中文字体

1. 在 `public/fonts/fonts_cn.css` 文件中添加新字体的 @font-face 定义：

```css
/* 新中文字体 */
@font-face {
  font-family: 'Your New Chinese Font';
  font-style: normal;
  font-weight: 400;
  font-display: swap;
  src: url('/fonts_cn/YourNewChineseFont-Regular.otf') format('opentype');
}
```

2. 在组件中修改字体类名

### 1.3 字体大小调整

在各个组件中修改字体大小类名：

- 标题大小：`text-4xl`、`text-5xl`、`text-6xl` 等
- 正文大小：`text-lg`、`text-base`、`text-sm` 等
- 小标签大小：`text-xs`

## 4. 项目详情页内容管理

### 4.1 创建新的项目详情页

1. 在 `components/projectContents/` 目录下创建新文件，例如 `p2.tsx`
2. 参考现有模板创建组件
3. 在 `components/projectContents/index.ts` 中注册新组件

### 4.2 修改现有项目详情页

1. 直接编辑对应的文件（如 `p1.tsx`）
2. 修改内容、布局或添加新的部分
3. 确保更新 `sections` 数组以反映导航变化

## 5. 常见调整场景

### 5.1 更改网站整体颜色主题

在各个组件中查找并修改以下颜色类：
- `text-stone-800`：主要文本颜色
- `text-stone-600`：次要文本颜色
- `text-stone-400`：辅助文本颜色
- `text-wabi-clay`：强调色
- `bg-wabi-paper`：背景色

### 5.2 添加新项目

1. 在 `constants.ts` 的 `PROJECTS` 数组中添加新项目数据
2. 准备项目图片并放入 `public/images/projects/` 目录
3. 如需自定义详情页，创建新的内容组件并注册

### 5.3 修改导航链接

在 `constants.ts` 文件中修改 `NAVIGATION_LINKS` 数组：

```tsx
export const NAVIGATION_LINKS = [
  { name: 'Portfolio', href: '#home' },  {/* 修改名称或链接 */}
  { name: 'Profile', href: '#profile' },
  { name: 'Projects', href: '#projects' },
  { name: 'Contact', href: '#contact' },
];
```

## 6. 注意事项

1. **图片优化**：确保所有图片都已优化，建议使用 WebP 格式以提高加载速度
2. **响应式设计**：修改内容时注意保持响应式布局，确保在各种设备上正常显示
3. **性能考虑**：避免使用过大的图片或过多的动画，以免影响网站性能
4. **备份**：在进行重大修改前，建议备份原始文件
5. **测试**：修改后在不同浏览器和设备上测试，确保兼容性
